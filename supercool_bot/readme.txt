Основные настройки находятся в начале файла main.py
Далее настройка уже через самого бота
Можно менять через бота в лс все настройки из config.json кроме chats_id (а именно интервал и текст)
А чтобы добавить новый чат в список чатов, нужно:
    Добавить бота в чат
    Дать ему админку
    Прописать это: /add_chat_to_list
Команды в боте в лс (работает только если акк админа)
    /start - просмотр текущих настроек
    /change setting value - вместо setting пишем настройку, вместо value - значение. Например:
        /change interval 15

Интервал в минутах
Файл pre.json нужен для удаления отправленных сообщений
